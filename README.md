# SauceDemo QA Testing Project

This repository contains manual QA testing performed on the SauceDemo website.

## Objective

The objective of this project manual testing techniques including:

- Test case creation
- Bug reporting
- Test execution
- Technical audit
- Evidence documentation

## Website tested

https://www.saucedemo.com/

## Test Scope

The following funcionalities were tested:

- Test login functionality
- Shopping cart behavior
- Product filters
- Checkout process
- Back button, session logout and refresh behavior
- Direct URL access to checkout page
- Duplicate checkout submissions

Technical Deep-Dive

- Cart's state management in LocalStorage
- Session hijacking/ expiration
- Network performance
